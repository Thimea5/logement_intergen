Stack Technologique :

back : PHP + MySql (bdd)


liste des commandes faite :
(le projet doit être sur wampserver, dans www/, c'est pour pouvoir héberger le back)
npm create vue@latest
cd front-end
npm install
npm install vuetify
npm install @mdi/font
npm install axios
npm install vue-router

lancer le projet en local, 
npm run dev
